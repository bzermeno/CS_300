# CS_300
This project involved developing a C++ program for managing course data. The application reads course information from a file, organizes it into an appropriate data structure, and provides users with options to view courses in alphanumeric order, display details (including prerequisites), and efficiently search for specific courses. The primary challenge was creating an efficient and user-friendly system to store, retrieve, and display course information while ensuring proper data organization and usability. The project required sorting, searching, and structured data storage to facilitate quick and accurate retrieval of course details. <br>
To overcome these roadblocks, I used debugging techniques such as print statements to verify data integrity and implemented a vector data structure and Course object structure to simplify data handling. Additionally, I reviewed best practices for sorting algorithms and ensured that prerequisites were mapped correctly. This project has improved the way I write code by emphasizing clarity, modularity, and documentation. By applying these principles, I have developed a scalable and adaptable program that can be enhanced with additional features in the future.<br>
